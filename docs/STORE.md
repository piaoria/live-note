## 2. 리스팅 문안 (복사용)

### 이름 (Name)
```
LiveNote — 유튜브 라이브 자막 타임라인
```

### 요약 (Short description, 132자 이내)
```
유튜브 라이브(주식방송) 자막을 Chrome 내장 AI로 정제해 [시간·종목·요약] 타임라인으로 기록합니다. 외부 전송 없이 온디바이스 처리.
```

### 상세 설명 (Detailed description)
```
LiveNote는 유튜브 라이브 방송, 특히 주식/투자 방송을 볼 때 유용합니다.

■ 이런 분께
- 방송 중간에 들어와 지난 맥락(어떤 종목 얘기가 오갔는지)을 빠르게 따라잡고 싶은 분
- 실시간으로 종목별 핵심 코멘트를 메모로 남기고 싶은 분

■ 동작 방식
1) 유튜브 라이브의 실시간 자동 자막을 읽어옵니다.
2) 30초/150자 단위로 묶고, 종목명·투자 키워드가 있는 부분만 골라냅니다.
3) Chrome 내장 AI(Gemini Nano)가 [종목명] 핵심 요약 형태의 한 줄로 정제합니다.
4) 영상 우측 사이드바에 [시간 + 종목 + 요약] 타임라인으로 쌓고, 복사/마크다운 다운로드를 제공합니다.

■ 개인정보 보호
- 외부 STT 서버나 외부 AI API를 사용하지 않습니다.
- 모든 요약은 사용자 기기 안에서(온디바이스) 처리되며, 데이터는 외부로 전송되지 않습니다.

■ 요구 사항
- Chrome 138 이상 (내장 Prompt API / Gemini Nano 지원 환경)
- 최초 사용 시 온디바이스 모델 다운로드가 필요할 수 있습니다.

※ 본 확장은 투자 자문이 아니며, 자막 요약의 정확성을 보장하지 않습니다. 투자 판단의 책임은 사용자에게 있습니다.
```
