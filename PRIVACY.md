# 개인정보 처리방침 (Privacy Policy) — LiveNote

_최종 수정일: 2026-06-24_

LiveNote(이하 "본 확장 프로그램")는 사용자의 개인정보를 존중하며, 아래와 같이 데이터를 처리합니다.

## 1. 수집·전송하지 않는 정보
본 확장 프로그램은 **어떠한 개인정보도 외부 서버로 수집·전송하지 않습니다.**
- 외부 STT 서버, 외부 LLM API(OpenAI/Anthropic 등) **를 사용하지 않습니다.**
- 자막 정제(요약)는 전적으로 **사용자 기기 내(온디바이스) Chrome 내장 AI(Gemini Nano)** 에서 수행됩니다.
- 분석 도구(애널리틱스), 광고 SDK, 추적기(tracker)를 포함하지 않습니다.

## 2. 로컬에 저장되는 정보
다음 데이터는 오직 사용자의 브라우저 내(`chrome.storage.local`)에만 저장되며, 기기를 떠나지 않습니다.
- 사용자 설정: 자막 숨김 여부, 추가 종목/키워드 목록.
- 생성된 타임라인: 영상별 `[타임스탬프 + 종목 + 요약]` 기록.

이 데이터는 사용자가 확장 프로그램을 삭제하거나 브라우저 데이터를 지우면 함께 삭제됩니다.

## 3. 권한 사용 사유
- `storage`: 위 설정 및 타임라인을 로컬에 저장하기 위함.
- 호스트 권한 `*://*.youtube.com/*`: 유튜브 페이지에서만 자막을 읽고 사이드바 UI를 표시하기 위함.
  유튜브 외 다른 사이트에서는 동작하지 않습니다.

## 4. 처리하는 콘텐츠
유튜브가 제공하는 자동 자막 텍스트를 페이지 내에서 읽어 요약에 사용합니다. 이 텍스트는
요약 처리를 위해 브라우저 내부에서만 사용되며 외부로 전송되지 않습니다.
